# Getting started

Awesome antd provides you improved fast reusable react components wrapped from antd components.

## Current Version

<Canvas withSource="none">Version: {version}</Canvas>

## Install

`npm install --save @galiojs/awesome-antd`

## Setup

Globally import components style.

```jsx
import '@galiojs/awesome-antd/es/style/component.css';
```
